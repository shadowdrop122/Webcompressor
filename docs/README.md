# WebCompressor 项目图表展示

本文件夹包含项目报告中需要用到的所有图表。

## 图表列表

| 文件 | 说明 | 预览方式 |
|------|------|----------|
| `diagrams.html` | 所有图表的HTML预览文件 | 双击在浏览器中打开 |
| `diagram1-arch.mmd` | 系统架构图 | Mermaid代码 |
| `diagram2-class.mmd` | 类图 | Mermaid代码 |

## 使用方法

### 方法1: 直接预览（推荐）
双击打开 `diagrams.html` 文件，在浏览器中查看所有渲染好的图表。

### 方法2: 在线编辑
访问 [Mermaid Live Editor](https://mermaid.live)，粘贴 `.mmd` 文件中的代码进行编辑和导出。

### 方法3: 导出图片
在 `diagrams.html` 页面中，右键点击任意图表 → "另存图片为..."

## 注意事项

- 推荐使用 Chrome/Edge 浏览器打开 `diagrams.html`
- 如果图表未显示，请确保网络连接正常（需要加载 Mermaid.js CDN）
